WWII Spy Thriller Mapbox Style

Created November 2019

Mapbox Studio

This vector Mapbox style was created for a cartography class. I had to make a style for a map that is to be used to promote a world war 2 spy movie. The primary inspiration was from army situation maps from world war 2 published by the Library of Congress. The colors and fonts were inspired by various movie posters.
